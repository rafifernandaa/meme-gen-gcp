# Serverless Meme Generator 
WIF Meme Generator 😼 is a fun, fully serverless web application that lets you create memes in seconds. Upload any image, type your own top/bottom text, or let Google Gemini AI auto-generate captions based on a topic and style you choose. The app supports multiple fonts (Impact, Barriecito, Birthstone), adjustable font sizes, and a randomizer for instant meme ideas.
Generated memes can be saved to a Google Cloud Storage gallery and shared directly via WhatsApp or X (Twitter). The app is containerized with Docker and deployed on Google Cloud Run for scalable, pay-per-use serverless hosting.

Tech Stack: Python · Streamlit · Pillow · Google Gemini API · Google Cloud Storage · Docker · Google Cloud Run
